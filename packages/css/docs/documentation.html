Named Colors

Contrast Pairing

This rule applies to all named families and semantic colours (error, warning, success). The -60 shade is the BASE — the primary shade and the contrast switch point for each family.

Shades below 60 are light backgrounds — use the -100 shade for text/icons on them. Shades above 60 are dark — use the -5 shade for text/icons.

The -60 shade is the switch point: depending on hue it meets 4.5:1 AA with either the -5 or -100 end of the palette.

Edge shades: -50 and -70 typically fail WCAG 4.5:1 AA for normal text but pass for large text (18px+) and decorative use (3:1).
